LookMyClass API

v1.0.0

LookMyClass is an EdTech SaaS platform for creating and managing academic curricula, AI-powered tests, documents, and subscriptions.

Authentication
Protected endpoints require a Bearer JWT token in the Authorization header.
Base URL
https://api.lookmyclass.com
Content Type
All requests use application/json unless otherwise noted.
Rate Limiting
API requests are rate-limited per user. Check response headers for current limits.

HTTP Methods

GET Retrieve resources
POST Create resources
PATCH Update resources
DELETE Remove resources
